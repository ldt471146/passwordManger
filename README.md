# 账号密码管理器（Electron）

本项目是一个离线优先的个人密码管理器，基于 Electron + 本地加密库实现，支持多主题、标签分组和导入导出。

## 功能
- 本地加密库（AES-256-GCM + PBKDF2）
- 搜索过滤、标签分组（支持多选）
- 标签库管理（颜色分类）
- 密码生成器
- 复制密码后自动清空剪贴板
- 导入/导出 JSON、导出 CSV
- 中文界面

## 运行
```bat
npm install
npm start
```

## 使用说明
1. 首次启动需要创建主密码
2. 主密码用于解锁本地库，忘记无法恢复
3. 标签用逗号分隔，例如：`工作, 个人, 金融`

## 目录结构
```
.
├─ main.js          # Electron 主进程
├─ preload.js       # 安全桥接
├─ renderer/        # 前端 UI
│  ├─ index.html
│  ├─ styles.css
│  └─ app.js
```

## 备注
- 库文件默认保存在系统用户目录下（通过“查看库路径”获取）
- 这是离线工具，不含云同步功能
